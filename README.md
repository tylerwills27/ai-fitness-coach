# AI Fitness Coach

AI-powered mobile fitness application that delivers conversational coaching, voice interaction, and personalized user settings in a cross-platform experience.

## Overview

AI Fitness Coach is a mobile application built to make fitness guidance more interactive and personalized than a traditional static app. It combines a React Native mobile frontend with a Python backend and AI-driven features to support conversational coaching, voice interaction, and saved user preferences.

## Features

- Conversational AI coaching
- Voice input and text-to-speech
- Personalized user settings
- Persistent preferences with SQLite
- Context-aware interaction flow
- Cross-platform mobile app experience

## Tech Stack

### Frontend
- React Native
- Expo

### Backend
- Python

### AI / Data
- OpenAI API
- LangChain
- SQLite
